# 1.概述

simple spring 是将项目中复杂的配置简化，开发难度降低,是一款快速开发的集成框架

# 2.特性

  • 1.基于ssm javaconfig 零xml配置
  • 2.常用的三层结构dao简化，service简化开发,controller层零开发
  • 3.依赖简单
  • 4.快速开发接口
  • 5.main函数直接启动
  • 6.多种应用场景实现
